Get Started
Quick Start

Up and running in
2 minutes

1

Clone the plugin

Download the SaleCraft plugin into your Claude Code project directory.

2

Configure MCP server

Add the Service System MCP endpoint to your Claude Code MCP configuration.

3

Start consulting

Run /mx in Claude Code and SaleCraft will greet you with a free marketing consultation.

terminal
# 1. Clone the repo
git clone https://github.com/MarinChen99097/
  Salecraft-Plugin.git

# 2. Open Claude Code and type:
/mx

# That's it. SaleCraft starts talking.
